description: Explain a file or function in plain English
---

Explain the following in plain, simple language: $ARGUMENTS

Include:
1. **What it does** — one-sentence summary
2. **How it works** — step-by-step walkthrough
3. **Why it exists** — the problem it solves
4. **Key dependencies** — what it relies on
5. **Gotchas** — non-obvious behavior or edge cases

Write as if explaining to a new team member.
